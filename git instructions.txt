1.git --version = for checking version of git.

2.git config --list = to check the configuration.

  git config --global user.name "your name"

  git config --global user.email "your email"

  ***these config is set for your pc, these don't effect if we change drives***

3.git init initializes a new Git repository in your current folder.

	When you run:

	git init

	Git creates a hidden folder named:

	.git

	This .git folder stores:

	commit history
	branches
	configuration
	tracking information
	all Git metadata

4. git status === this shows any new files, modified files, staged files.

5. git add . === stages all files for next commit.

6. git commit -m "type your msg "

TO CHANGE NAME OF BRANCH WE USE:

1. git branch -m ***name***

After this we need to push while using we may get upstream error ,so we need set remote as upstream use 
git push -u origin *** name***

2. git branch === gives list of branches.

3. git checkout -b ***name*** === creates new branch with name and switches to it.

4. git branch -d ***hero*** === deletes branch with name "hero" , if it is not current branch.

5. git checkout ***hero*** === switches to branch with name "hero".
